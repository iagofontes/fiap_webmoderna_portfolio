Tecnologias

Aqui estão algumas das tecnologias com as quais eu já tive algum contato:

Com o JavaScript trabalhei durante um bom tempo, tanto no front=end quanto no back-end.

Com o TypeScript trabalhei durante um curto período, auxiliando num projeto com Angular 4.

O Nodejs foi um grande companheiro, durante a construção de um WebService.

Já com Ionic, nunca tive a oportunidade de desenvolver algo com, porém fiz alguns cursos trabalhando com a versão 2.

Angular foi um grande companheiro, durante alguns meses, assim como o Nodejs. Trabalhei com as versões 2 e 4.

Java é algo que todo programador viu, com certeza. Comigo não foi diferente, porém nunca implementei nenhum sistema com a linguagem, sempre utilizei somente para estudos.

O PHP consome uma boa parcela da web, desde que comecei a trabalhar com web sempre utilizei PHP para realizar o processamento dos dados no back-end.

Quando comecei a trabalhar com programação, o Delphi foi a primeira linguagem com que tive contato. Uma linguagem relativamente fácil e intuitiva, porém sem muita inovação.

Experiências

Minhas experiências profissionais são breves, trabalho em meu primeiro emprego até os dias atuais, porém já trabalhei com as diversas tecnologias citadas acima. Logo abaixo apresento um pouco sobre a empresa.

A Urgtec é uma empresa que cria software para laboratórios de protese dentária. A empresa encontra-se no mercado por volta de 30 anos, atendendo ao ramo de laboratórios protéticos. Atualmente nos encontramos na fase de transição, partindo para desenvolvimento de aplicações mobile e WebServices para gerenciar as requisições.

Por fim, estamos em fase de constante crescimento. Atualmente a empresa é líder de mercado.
